### Requirements :wrench:
- Node.js >= v7.0.0
- NPM >= 3.10.8

### How to Run :runner:
first you need to start the server that feeding our application.
in api-server file, run:
- npm start or node server.js

in readble file, run:
- npm install
- npm start
